# Brand Designs & Assets Directory

This directory is the official source of truth for the visual identity of **Lashes & MGlamour**.

---

## 📁 Directory Structure

```text
designs/
├── logo/               # Official logo files (Lashes = Gold, MGlamour = Black)
├── flyers/             # Salon flyers and handouts
├── covers/             # Facebook & social media covers
├── posts/              # Instagram posts & social graphics
├── business_cards/     # Business card designs
├── trifold/            # Service trifold brochures
├── menus/              # Service menu assets
├── website/            # Website design mockups and digital UI layouts
└── advertising/        # General advertising & promotional materials
```

---

## ⚠️ Important Rules

1. **Logo Protection**:
   - The files in `designs/logo/` must not be modified or reinterpreted.
   - Gold Hex code for "Lashes" is `#D4AF37`.
   - Black Hex code for "MGlamour" is `#0B0B0B`.
2. **Identity Source**:
   - Every styling component, token, or color palette defined in the application code must match the assets located in this directory. Do not introduce new brand styles.
